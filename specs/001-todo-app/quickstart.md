# Quickstart

This guide provides instructions on how to run the Todo application.

## Prerequisites

- Python 3.12

## Running the Application

1.  Navigate to the root directory of the project.
2.  Run the application from the command line:

    ```bash
    python src/main.py 
    ```
    *(Note: The main entry point `main.py` does not exist yet, but will be created during implementation.)*
